# Contributing to GeneAtlas

Thank you for considering contributing to **GeneAtlas** — a project with the potential to reshape humanity's relationship with its own biology.

## Code of Conduct

We follow the [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/).  
Be respectful, inclusive, and constructive.

## How Can You Contribute?

### 1. Bug Reports & Feature Requests

- Open an issue using the appropriate template
- Clearly describe the problem or suggested feature
- Include screenshots or mockups when possible

### 2. Code Contributions

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-idea`)
3. Make your changes
4. Test thoroughly
5. Submit a Pull Request

### 3. Areas We Need Help With

- **Frontend Development** (UI/UX, animations, 3D visualization)
- **Design** (logos, illustrations, scientific visualizations)
- **Bioinformatics** (data models, genomics algorithms)
- **AI/ML** (model training pipelines, evaluation)
- **Security & Privacy** (encryption, zero-knowledge proofs)
- **Documentation** (whitepapers, API docs)
- **Ethics & Policy** (frameworks for responsible use)

### 4. Scientific & Research Contributions

- Suggest relevant papers
- Help validate biological accuracy
- Review phenotype prediction models
